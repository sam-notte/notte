---
title: "wait"
description: "Waits for the specified agent to complete"
---





## Parameters

<ParamField path="agent_id" type="str">
  The identifier of the agent to wait for.
</ParamField>

<ParamField path="polling_interval_seconds" type="int">
  The interval between status checks. (default: 10)
</ParamField>

<ParamField path="max_attempts" type="int">
  The maximum number of attempts to check the agent's status.  (default: 30)
</ParamField>

## Returns

[`AgentStatusResponse`](/sdk-reference/misc/agentstatusresponse): The response from the agent status check.
